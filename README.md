## Description

A password generator app. 

## Installation
# Web
```sh
1. Clone the repo
2. cd into the directory
3. Run npm install
4. Run ng serve -o
```
# Electron
```sh
1. Clone the repo
2. cd into the directory
3. Run npm install
4. Run npm run electron
```
